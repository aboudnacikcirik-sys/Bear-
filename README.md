# Bear Puzzle Escape

Небольшая браузерная игра, вдохновлённая Roblox Bear Alpha: игрок-стикман должен решить три пазла, открыть ворота и не попасться медведю.

## Как запустить

Открой `index.html` в браузере или подними локальный сервер:

```bash
python3 -m http.server 8000
```

После этого перейди на `http://localhost:8000`.

## Управление

- `WASD` или стрелки — движение стикмана.
- `E` — чинить ближайший пазл или выйти через открытые ворота.
- `Начать заново` — перезапуск раунда.

## Цель

1. Найди три светящихся пазла.
2. У каждого пазла нажми `E` три раза рядом с ним.
3. Когда ворота сверху карты станут зелёными, подойди к ним и нажми `E`.
4. Не дай медведю догнать стикмана.
